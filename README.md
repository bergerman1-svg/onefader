# OneFader

פיידר אחד. שליטה חלקה בווליום של המחשב — למורות, מנהלות הצגה ובימאים.

## התקנה והרצה

**Mac:** דאבל-קליק על `START_HERE_MAC.command` (בפעם הראשונה: קליק ימני ← Open בגלל Gatekeeper). הוא מתקין הכל לבד ומריץ.

**Windows:** דאבל-קליק על `START_HERE.bat`.

להפצה למשתמשות: `BUILD_APP_MAC.command` (מק) או `BUILD_EXE.bat` (ווינדוס) — מייצרים אפליקציה אחת שרצה בלי Python.

### הרצה ראשונה של הגרסה המופצת (אפליקציה לא חתומה)

- **Mac:** קליק ימני על `OneFader.app` ← **Open** ← ושוב **Open** בחלון. פעם אחת בלבד.
- **Windows:** בסרגל ההורדות של הדפדפן בחר **Keep/שמור**. אם קופץ "Windows protected your PC" ← **More info** ← **Run anyway**.

זה קורה כי האפליקציה לא חתומה בתעודת Code Signing — לא תקלה. השלבים חד-פעמיים ורק אומרים למערכת שאתה בוטח בקובץ.

### או ידנית (Windows)

1. להתקין Python מ-python.org (לסמן "Add to PATH" בהתקנה).
2. בתיקיית הפרויקט, ב-Command Prompt:

```
pip install pywebview pycaw comtypes websockets qrcode pillow
python onefader.py
```

## שליטה מהטלפון (Remote)

1. לוחצים על אייקון הטלפון בפינת האפליקציה.
2. סורקים את ה-QR עם מצלמת הטלפון (המחשב והטלפון על אותו WiFi).
3. מקלידים את ה-PIN שמופיע על מסך המחשב — ויש פיידר ביד.

עובד על כל אייפון ואנדרואיד, בלי להתקין כלום — רק דפדפן. הטלפון מתחבר מחדש אוטומטית אם ה-WiFi מנתק לרגע.

## שימוש

- **גרירת הפיידר** — שליטה ידנית מיידית (מבטלת פייד פעיל).
- **FADE IN / FADE OUT** — פייד אוטומטי חלק לרמה האחרונה / לאפס.
- **3s / 5s / 8s** — בחירת משך הפייד.
- **מקש רווח** — פייד חכם: אם יש סאונד → פייד אאוט, אם שקט → פייד אין.
- **מקשים 1 / 2 / 3 / 4** — פייד חלק ל-25% / 50% / 75% / 100%.

## שלט MIDI

לוחצים על אייקון ה-MIDI (עיגול עם 5 נקודות) בפינת האפליקציה:

1. מחברים שלט MIDI (פיידר/נוב/כפתורים) ב-USB.
2. לוחצים **LEARN** ליד התפקיד הרצוי (Volume / Fade In / Fade Out).
3. מזיזים את הפיידר או לוחצים על הכפתור בשלט — וזהו, מחובר.

החיבורים נשמרים (`~/.onefader-midi.json`) וחלים גם אחרי הפעלה מחדש. אפשר לחבר/לנתק שלט תוך כדי ריצה.
- **Esc או לחיצה על שורת הסטטוס** — עצירת פייד באמצע.
- **חיצים למעלה/למטה** (כשהפיידר בפוקוס) — כוונון עדין. עם Shift — צעדים גדולים.

## בניית EXE אחד להפצה (בלי Python אצל המשתמשת)

```
pip install pyinstaller
pyinstaller --onefile --windowed --name OneFader --add-data "ui.html;." --add-data "remote.html;." onefader.py
```

הקובץ ייווצר ב-`dist\OneFader.exe` — קובץ אחד שרץ בדאבל-קליק.

## הערות

- האפליקציה שולטת ב-Master Volume של Windows, כך שהיא עובדת עם כל נגן — Spotify, YouTube, VLC, כל דבר.
- עקומת הפייד היא S-curve חלקה (לא לינארית) כדי שהפייד יישמע טבעי לאוזן.
- אם הווליום היה על Mute, גרירת הפיידר או פייד אין מבטלים אותו אוטומטית.
